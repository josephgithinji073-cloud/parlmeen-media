## Component Colors

### Buttons
- Default: `linear-gradient(135deg, #1e40af, #0369a1)`
- Hover: `linear-gradient(135deg, #3b82f6, #06b6d4)`
- Shadow: `rgba(59, 130, 246, 0.4)`

### Header
- Background: `linear-gradient(90deg, rgba(30, 64, 175, 0.9), rgba(3, 105, 161, 0.85))`
- Border: `2px solid #00d9ff`

### Forms
- Input Border Focus: `2px solid #3b82f6`
- Input Shadow Focus: `0 0 15px rgba(59, 130, 246, 0.4)`
- Success Message: `color: #10b981; background: rgba(16, 185, 129, 0.15)`

## Quick Usage

```css
/* Buttons */
button { background: var(--primary); }

/* Headings */
h2 { color: var(--primary-light); }

/* Text */
p { color: var(--text-secondary); }

/* Accents */
border { border-color: var(--accent); }

/* Success/Error */
.success { color: var(--success); }
.error { color: var(--error); }
```
